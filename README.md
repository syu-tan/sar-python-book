# sar-python-book




## 環境構築

### 方法 1.

Python環境管理ツール[Anaconda](https://www.anaconda.com/download) を 使用する場合

```shell
# 仮想環境作成
conda create -n sar-book python=3.10

# 仮想環境適用
conda activate sar-book

# Gdal環境
conda install -c conda-forge gdal

# 基礎環境構築
pip install -r env/requirements.txt
```
